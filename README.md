PhotoMaker
==========

Веб-сервис для автоматической генерации маркетплейс-фотографий одежды на основе:
- фото-референса модели;
- PNG-футболки с уже нанесённым принтом;
- промпта, сгенерированного через OpenAI для Nano Banana (Kie.ai).

Технологический стек (v1):
- Python 3.11+
- Flask + SQLAlchemy + Jinja2
- SQLite

Структура проекта:
- `app/` — код приложения (роуты, сервисы, интеграции, шаблоны, статика).
- `data/` — файлы (референсы, футболки, результаты, архивы, временные файлы).
- `database/` — SQLite база.
- `logs/` — логи приложения и интеграций.
- `workers/` — фоновые воркеры для генераций и, при необходимости, промптов.
- `scripts/` — служебные скрипты (синхронизация, превью, клинап).

Запуск (черновой, для разработки):
1. Установить зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Создать `.env` на основе примера из `README.md`.
3. Запустить приложение:
   ```bash
   python run.py
   ```
